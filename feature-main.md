# 算法实验室
这是一个有springboot+vue3实现的算法实验室，用于展示和演示算法的实现。具体包括代码演示，动画演示，时间复杂度和空间复杂度分析等。
## 项目初始化
1. 创建alg-lab-backend后端项目，使用springboot初始化项目，添加必要的依赖，包括springboot web，springboot data jpa，mysql driver，lombok等。
2. 创建alg-lab-frontend前端项目，使用vue3初始化项目，添加必要的依赖，包括vue3，vue router，element-plus等。
3. 在README.md完善介绍，并且需要按照算法的分类，将所有的算法列举出来，包括不限于，AI算法，数据结构算法，排序算法，搜索算法，动态规划算法，贪心算法，分治算法，回溯算法，图算法，字符串算法等,特殊算法，通用算法，并查集，红黑树，跳表，堆排序等，最小生成树，图论等等，需要尽可能的详细。
4. 然后根据这些算法的名字，创建不同的分支，目前并不需要实现。
5. 不需要用户登陆，直接在首页展示所有的算法，点击不同的算法，展示该算法的代码演示，动画演示，时间复杂度和空间复杂度分析等。通过二级下拉菜单来展示不同的算法，就以冒泡排序为例子实现一个样例。如果效果好的话，其他算法也可以按照这个例子来实现。
6. 分支命名的规则就是feature-算法名，比如feature-bubble-sort.
7. 要求界面美观大气，有黑科技的风格。
8. 最后将代码提交到feature-main分支，其他所有分支的代码都是基于feature-main分支的。
9. 需要保障feature-main分支的代码是稳定的，并且能正常运行的，本地的数据库是mysql,端口3306,用户名和密码都是root,redis端口6379,没有密码。
